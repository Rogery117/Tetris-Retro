# Tetris-Retro 🕹️

Un juego de Tetris clásico desarrollado en **Ensamblador (x86)** para entornos DOS. Este proyecto busca recrear la experiencia de los juegos de finales de los 80, optimizando la lógica de memoria y el manejo de gráficos en 16 bits.

## 🚀 Características
*   **Lógica de Juego:** Implementación completa de gravedad, rotación de piezas y eliminación de líneas.
*   **Gráficos:** Modo de video estándar de DOS para una estética retro auténtica.
*   **Optimización:** Escrito en **TASM** (Turbo Assembler) centrado en el uso eficiente de registros y segmentos de memoria.

## 🛠️ Tecnologías utilizadas
*   **Lenguaje:** Assembly (ASM) x86.
*   **Entorno:** DOSBox / MS-DOS.
*   **Herramientas de desarrollo:** TASM, VS Code.

## 📁 Estructura del Proyecto
*   `/src`: Contiene el código fuente principal del juego.
*   `/docs`: Documentación técnica del proyecto (opcional).
